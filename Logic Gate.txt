Berikut adalah penjelasan untuk 7 gerbang logika dasar berdasarkan prinsip Dasar Elektronika dan Teknik Digital:
1. Gerbang AND: Gerbang AND akan menghasilkan output logika 1 hanya jika seluruh atau kedua inputannya bernilai 1. Jika di salah satu inputan atau keduanya bernilai 0, maka outputnya akan bernilai 0.
2. Gerbang OR: Gerbang OR akan menghasilkan output logika 1 jika di salah satu inputan atau kedua inputannya bernilai 1. Gerbang OR hanya akan menghasilkan output logika 0 jika seluruh inputannya bernilai 0.
3. Gerbang NOT: Gerbang NOT berfungsi sebagai pembalik (inverter). Jika inputan bernilai 1, maka output akan bernilai 0, dan jika inputan bernilai 0, maka output akan bernilai 1.
4. Gerbang NAND: Gerbang NAND adalah kebalikan dari gerbang AND. Output Gerbang NAND akan bernilai 0 hanya jika seluruh atau kedua inputannya bernilai 1. Jika di salah satu inputan atau keduanya bernilai 0, maka outputnya akan bernilai 1.
5. Gerbang NOR: Gerbang NOR adalah kebalikan dari gerbang OR. Output Gerbang NOR akan bernilai 1 hanya jika seluruh atau kedua inputannya bernilai 0. Jika di salah satu atau kedua inputannya bernilai 1, maka outputnya akan bernilai 0.
6. Gerbang XOR (Exclusive OR): Gerbang XOR akan menghasilkan output logika 1 jika kedua inputannya memiliki nilai yang berbeda (misal A=1, B=0). Jika kedua inputannya bernilai sama (keduanya 0 atau keduanya 1), maka outputnya akan bernilai 0.
7. Gerbang XNOR (Exclusive NOR): Gerbang XNOR adalah kebalikan dari gerbang XOR. Output akan bernilai 1 jika kedua inputannya memiliki nilai yang sama (keduanya 0 atau keduanya 1). Jika kedua inputannya berbeda, maka outputnya akan bernilai 0.
